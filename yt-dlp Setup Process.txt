1: Download yt-dlp using this command " pip install -U yt-dlp " 
2: Install Chrome Extension to download Cookies. Any one will work but I used this one: download here: https://chromewebstore.google.com/detail/get-cookiestxt-locally/cclelndahbckbenkjhflpdbgdldlbecc
3: download ffmpeg.z file from here: https://www.gyan.dev/ffmpeg/builds/
4: Extract the ffmpeg file in the same location 

To download Whole Playlist

yt-dlp -f "bestvideo+bestaudio/best" --merge-output-format mp4 --ffmpeg-location "E:\yt-dlp Setup\ffmpeg\bin" --cookies cookies.txt "https://youtu.be/vnm2yd1f-a4?list=PLmgyxPj-5jn5aB1EBtGnui2aSFLB_D2nX&t=9576"


To Download a single video OR a single video  from playlist:

yt-dlp -f "bestvideo+bestaudio/best" --merge-output-format mp4 --ffmpeg-location "E:\yt-dlp Setup\ffmpeg\bin" --cookies cookies.txt --no-playlist "https://www.youtube.com/watch?v=dL4Vn4hKxQ4"
